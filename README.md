# 訪問内容一覧 モバイルページ

訪問内容一覧表.xlsx をスマホ閲覧用の1枚のHTMLに変換し、Vercelで公開する仕組み。

## ファイル構成

- `convert.py` — xlsx → index.html 変換スクリプト
- `template.html` — ページの見た目(デザイン変更はここ)
- `index.html` — 生成物(これが公開される)
- `middleware.js` — Basic認証(Vercel Edge Middleware)

## 初回セットアップ

```bash
vercel link          # 新規プロジェクトとして紐付け
vercel env add BASIC_AUTH_USER   # 例: tassha
vercel env add BASIC_AUTH_PASS   # ヘルパーに共有するパスワード
vercel --prod
```

発行されたURLを LINEグループのノートに固定 → 完了。
※ パスワード未設定(空)の場合は全アクセスを拒否する安全側の仕様です。

## 内容に変更があったとき(普段の運用)

1. Excelを編集して保存
2. ターミナルで:

```bash
python convert.py 訪問内容一覧表.xlsx
vercel --prod
```

URLは変わらないので、LINEには「○○さんの内容を更新しました」と一言流すだけ。

## Excel側のルール(変換が正しく動く条件)

- シート名 = 利用者名(画面にはシート名が表示される。氏名セルB3は見ない)
- B1に「【援助方針】」、C1に方針本文
- 2行目がヘッダー(氏名/曜日/時間/サービスコード/支援内容)
- 曜日セル(C列)が空の行は直前の曜日の2件目として扱う
- B列の「※」で始まる行は注意事項としてオレンジ枠に表示

## 画面の仕様

- 利用者をプルダウンで選択、曜日タブで切替(今日の曜日に赤丸・初期選択)
- 訪問がない曜日はタブがグレー表示
- 援助方針は折りたたみ、※注意事項は常時表示
- 検索エンジンには noindex 指定済み
